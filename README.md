# Hyrox Explorer

   A data visualization project exploring ~90k Hyrox race results.
   Built with React, TypeScript, Tailwind, and Recharts.

## Data source

  Race results dataset from [Kaggle](https://www.kaggle.com/datasets/jgug05/hyrox-results),
  originally scraped from the public Hyrox results site. Contains ~90k individual
  race results across multiple seasons, with per-segment splits for all 8 runs,
  8 stations, and transitions.

## Running locally

```bash
   npm install
   npm run dev
```
